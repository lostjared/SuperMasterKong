# SuperMasterKong
Port of a video game I wrote in my early twenties. 


Build and run the game:
```bash
mkdir build && cd build
cmake ..
make -j$(nproc)
cp SuperMasterKong ../SuperMasterKong
cd ../SuperMasterKong
./SuperMasterKong
```
